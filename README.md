# Enriquecimento Instrumental (UFU) 🎓🤖

Este repositório contém os projetos e experimentos desenvolvidos durante a disciplina de **Enriquecimento Instrumental** no 1º período de Engenharia de Computação na Universidade Federal de Uberlândia (UFU).

O objetivo central da disciplina foi estabelecer a base prática em eletrônica e sistemas embarcados, focando no controle de componentes e lógica de programação aplicada ao hardware.

---

### 🛠️ Tecnologias e Componentes Utilizados
* **Microcontrolador:** Arduino (Uno/ESP8266)
* **Linguagem:** C++ (Arduino Framework)
* **Componentes:** Sensores de luz (LDR), LEDs RGB, Sensores de temperatura/umidade, Módulos Wi-Fi.

### 📚 O que foi desenvolvido
Durante as aulas e laboratórios, explorei conceitos fundamentais como:
* **Circuitos Elétricos:** Montagem e análise de circuitos em protoboard.
* **Lógica de Controle:** Manipulação de entradas e saídas digitais/analógicas.
* **Automação Wi-Fi:** Integração de hardware com redes sem fio para controle remoto (Ex: `LedRGBwifi`).
* **Sistemas de Sensoriamento:** Leitura e processamento de dados de sensores externos.

### 📂 Estrutura de Pastas
* `/códigos`: Contém os sketches (.ino) organizados por aula e projeto final.
    * `LedRGBwifi`: Controle de iluminação via rede.
    * `aula06-luznoturna`: Implementação de sistema automático baseado em luminosidade.
    * `projeto_final`: Protótipo integrador de final de semestre.
